These above are simple sortable lists which, again (as table example), require a special plugin, but still you can make it work with a few lines of code. Here is the code of the second list: 
$('.sortlist').find('li').draggable({
    container: 'parent',
    placeholder: true,
    target: $('.sortlist').find('li'),
    cls: {placeholder: 'liholder'},
    targetOptions: {
        tolerance: 'pointer',
        over: function(e) {
            if (e.placeholder.next().get(0) == e.self[0]) return true;
            e.placeholder.insertBefore(e.self);
        },
        drop: function(e) {
            if (e.draggable.next().get(0) == e.self[0]) return true;
            e.draggable.insertBefore(e.self);
        }
    }
});
